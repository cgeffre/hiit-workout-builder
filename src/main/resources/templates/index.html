Welcome

This application allows you to build and then be led through custom high intensity interval training (HIIT) workouts. HIIT workouts are relatively short and typically combine strength and cardiovascular exercises for an intense training session. Use the above navigation to create a list of exercises, add the exercises of your choice to a workout program, and then be led through the workouts that you create. You can read more about HIIT on Wikipedia. This application is under active development with new features being added regularly; you can always find the most recent version the GitHub repository for this project.
